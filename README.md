# Sales Management System

## Overview

This project is a Sales Management System built using SQL Server and Python.

The project demonstrates database design, data automation, and SQL querying.

## Features

* Relational Database Design
* Primary Keys & Foreign Keys
* Customer, Product, and Order Management
* Python Automation using pyodbc
* Sample Data Generation
* SQL Queries and Reports

## Technologies

* SQL Server
* Python 3
* pyodbc
* Git & GitHub

## Project Files

* `sales_db.sql` → Database creation script.
* `analysis_sales.py` → SQL analysis queries.
* `to prepare data insertion.py` → Python automation for data generation and insertion.
* `Sales_Management_System_Project.pdf` → Project documentation.

## Learning Outcomes

Through this project, I practiced:

* Database Design
* SQL Joins
* Foreign Keys
* Data Automation
* Python & SQL Server Integration
* Problem Solving
